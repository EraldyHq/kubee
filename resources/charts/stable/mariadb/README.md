# Kubee Mariadb Chart

This `kubee chart` installs the [MariaDb Operator](https://github.com/mariadb-operator/mariadb-operator).

## Installation

* In your cluster values file
```yaml
mariadb:
    enabled: true
```
* Play
```bash
kubee -c clusterName helmet play mariadb
```
