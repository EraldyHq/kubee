

You can install the [cert manager command line (cmctl)](https://cert-manager.io/docs/reference/cmctl/#installation)
```bash
brew install cmctl
```

You can check the api via:
```bash
cmctl check api
```
```
The cert-manager API is ready
```


See also: [How to use it with Traefik](https://doc.traefik.io/traefik/user-guides/cert-manager/)
